ESG Classification Web API Help

Введение

Добро пожаловать в документацию нашего проекта! Мы используем кастомную модель, построенную на основе MPNet, для классификации ESG (экологических, социальных и управленческих) рисков из текстовых данных. В последнее время компании уделяют ESG большое внимание! А вдруг следование принципам ESG линейно влияет на капитализацию компании?! Модель использует среднее пуллинг и кастомную классификационную голову для получения оценок ESG-рисков. Модель ESGify основана на архитектуре MPNet и включает кастомную классификационную голову, адаптированную для классификации ESG-рисков. Модель выдает логиты для нескольких категорий ESG-рисков, которые затем преобразуются в вероятности с помощью сигмоидной функции активации. Ниже представлены ссылки на использование нашей модели.
